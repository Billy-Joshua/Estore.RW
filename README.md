## 🐛 Troubleshooting

| Issue | Cause | Solution |
|-------|-------|----------|
| Images not displaying | Wrong paths or missing files | Check `images/` folder, verify filenames |
| Styles look incorrect | Browser cache | Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac) |
| Form submissions fail | JavaScript not loaded | Check console (F12), verify script tags |
| Cart not persisting | Private/Incognito mode | Disable private mode, check localStorage enabled |
| Mobile layout broken | Viewport meta tag missing | Already included - check for CSS conflicts |
| WhatsApp links inactive | Invalid phone format | Use +250XXXXXXXXX (Rwanda format) |
| Slow performance on mobile | Heavy animations | Already optimized - check network throttling |

**Debug Tips**: Open DevTools (F12) → Console tab to see detailed error messages.

### Image Optimization Tips:
- Size: 300x250px for products, 100x100px for logo
- Format: JPEG for photos, PNG for graphics
- Quality: 70-85% compression
- File size: Max 500KB per image
- Aspect ratio: 1:1 or 4:3 for consistency

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| Images not showing | Check filenames match (case-sensitive), check `images/` path |
| Styles look broken | Clear cache (Ctrl+Shift+Del), hard refresh (Ctrl+F5) |
| Forms not working | Check console (F12), check JavaScript loaded |
| Cart not saving | Disable private/incognito mode, enable localStorage |
| Mobile looks off | Check viewport meta tag, test on actual device |
| WhatsApp links broken | Check phone number format (+250...), test link |
